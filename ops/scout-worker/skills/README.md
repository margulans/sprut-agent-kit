# Scout Skills

Каталог скиллов внешнего агента `Scout`:

- `web-search` — быстрый поиск в сети.
- `youtube-search` — поиск видео и каналов YouTube.
- `social-search` — поиск по X/LinkedIn/Reddit.
- `messenger-channels` — поиск каналов Telegram/Discord.
- `deep-research` — глубокий ресерч по теме.
- `reddit-scan` — анализ обсуждений Reddit.
- `youtube-transcribe` — транскрипция и summary YouTube.
- `summarize` — универсальный саммаризатор.
- `analyze-topic` — анализ темы (web + reddit).
- `orchestrator` — сценарии сквозного ресерча.

Публичный task-каталог доступен через `task_type = "skills_catalog"`.
